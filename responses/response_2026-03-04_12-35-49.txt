 To create a Python function that removes all non-alphanumeric characters from a string while maintaining the integrity of existing code, you can use the `re` module for regular expressions. This approach is efficient and leverages Python's built-in capabilities. Below is the implementation of such a function along with brief explanations on its logic:

```python
import re

def clean_string(input_str):
    """
    Removes all non-alphanumeric characters from the input string.
    
    Args:
        input_str (str): The input string to be cleaned.
        
    Returns:
        str: A string with only alphanumeric characters and spaces.
    """
    # Use regex to replace anything that is not a letter, number, or space with an empty string
    clean_str = re.sub(r'[^a-zA-Z0-9\s]', '', input_str)
    return clean_str
```

### Explanation:
1. **Import the `re` module**: This module provides support for regular expressions in Python. Regular expressions are patterns used to match and manipulate text.
2. **Define the function `clean_string(input_str)`**: The function takes a single argument, `input_str`, which is expected to be a string.
3. **Use `re.sub` for substitution**: The `re.sub` function is used with a pattern that matches any character not in the set of alphanumeric characters (`a-zA-Z0-9`) and spaces (`\s`). This pattern is enclosed within single quotes after the `r` prefix, which denotes a raw string (useful for regular expressions to avoid issues with escape sequences).
4. **Return the cleaned string**: The function returns the modified string where all non-alphanumeric characters have been removed.

### Integration into Existing Code:
To integrate this function seamlessly into your existing codebase without disrupting its functionality, you would call `clean_string` wherever necessary for cleaning strings. For example, if there's a need to clean the column names or specific fields in your DataFrame, you can apply this function as follows:

```python
import pandas as pd

# Example usage within a larger script
df = pd.DataFrame({'data': ['Hello!', 'World?', '123-abc']})
df['cleaned_data'] = df['data'].apply(clean_string)
print(df)
```

This example demonstrates how to apply the `clean_string` function to a DataFrame column, ensuring that all strings in the specified column are cleaned of non-alphanumeric characters.
